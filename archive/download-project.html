🌐 方法1: GitHub Web UI（推奨）

  1. GitHubリポジトリ作成
    • GitHub新規リポジトリ作成
    • Repository name: pocketnavi_02
    • Public/Private を選択
    • 「Create repository」をクリック
  2. ファイルアップロード
    • 「uploading an existing file」をクリック
    • プロジェクトファイルを全選択してドラッグ&ドロップ
    • Commit message: "Initial commit with Supabase integration"
    • 「Commit changes」をクリック

💻 方法2: ローカル環境でGit操作

Step 1: ローカルにプロジェクト作成

mkdir pocketnavi_02

git init

Step 2: ファイルをローカルにコピー

WebContainerからダウンロードしたファイルを配置

🔧 環境変数の設定

重要: .env ファイルはGitHubにアップロードしないでください。

代わりに .env.example ファイルを作成：

# Supabase設定
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
VITE_USE_SUPABASE=true

✅ 完了後の確認

  • ✅ 全ファイルがアップロードされている
  • ✅ .env ファイルが除外されている
  • ✅ README.md が適切に表示されている
  • ✅ package.json の依存関係が正しい
